# DSA - Data Structures & Algorithms

A collection of C++ implementations for data structures and algorithms, organized by week and topic.

## Structure

- **week1-4** - Weekly problem sets and implementations
- **01-sliding-window** - Sliding window pattern problems
- **sort-search** - Sorting and binary search implementations
- **patterns.md** - Algorithm patterns reference

## Topics Covered

- Linked Lists (insert, delete, reverse, merge)
- Queues & Stacks (implementation and applications)
- Searching (binary search, linear search)
- Sorting & Comparators
- BFS/Multi-source BFS
- Sliding Window Techniques
- Post-fix Expressions

## Building & Running

Compile with:
```bash
clang++ -std=c++17 -stdlib=libc++ file.cpp -o output
```

Or use the build task in VS Code (Cmd+Shift+B).
