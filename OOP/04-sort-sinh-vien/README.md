# Sắp xếp danh sách sinh viên

## Mô tả

Chương trình quản lý danh sách sinh viên đơn giản bằng C++.

Mỗi sinh viên gồm các thông tin:

- Mã sinh viên
- Họ tên
- Lớp
- Email

Chương trình nhập danh sách sinh viên đến khi hết dữ liệu đầu vào, sau đó sắp xếp danh sách theo mã sinh viên tăng dần và in kết quả ra màn hình.

## Kiến thức sử dụng

- Class và object
- Constructor
- Friend function
- Nạp chồng toán tử `>>`
- Nạp chồng toán tử `<<`
- Nạp chồng toán tử `<`
- `vector`
- `std::sort`
