# Quản Lý Sinh Viên (Student Management)

## Mô tả
Ứng dụng quản lý sinh viên với hỗ trợ phân loại: sinh viên cao đẳng và sinh viên đại học.

## Cấu trúc dự án
- `SinhVien.h / SinhVien.cpp` - Lớp cơ sở cho sinh viên
- `SinhVienCaoDang.h / SinhVienCaoDang.cpp` - Lớp sinh viên cao đẳng
- `SinhVienDaiHoc.h / SinhVienDaiHoc.cpp` - Lớp sinh viên đại học
- `main.cpp` - Chương trình chính

## Tính năng
- Tạo và quản lý danh sách sinh viên
- Phân loại sinh viên theo trình độ đào tạo
- Hiển thị thông tin sinh viên
